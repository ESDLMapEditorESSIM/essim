post /simulation

Create and start a simulation run

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • esdlContents: required(string)
  • user: required(string)
  • scenarioID: required(string)
  • simulationDescription: required(string)
  • startDate: required(datetime)
  • endDate: required(datetime)
  • timeStep: required(object)
    • value: required(number)
    • unit: required(one of SECOND, MINUTE, HOUR, DAY, WEEK, MONTH, YEAR)
  • grafanaURL: (string)
  • influxURL: (string)
  • natsURL: (string)
  • mqttURL: (string)
  • kafkaURL: (string)
  • externalSolvers: (array of type.ExternalSolver)

    Items: ExternalSolver

    • name: required(string)
    • url: required(string)
    • orderPreference: (one of FIRST, LAST)
  • kpiModule: (object)
    • kafkaURL: required(string)
    • modules: required(array of type.KPIModule)

      Items: KPIModule

      • id: required(string)
      • result: (string)
      • config: required(object)
    • simRunDate: (datetime)
    • transport: (array of type.TransportNetwork)

      Items: TransportNetwork

      • name: required(string)
      • networkHTMLDiag: required(string)
    • dashboardURL: (string)
    • loadAnimationJSON: (string)
    • nodeConfig: (array of type.NodeConfiguration)

      Items: NodeConfiguration

      • esdlNodeId: required(string)
      • remoteNodeLogic: required(boolean)
      • mqttHost: required(string)
      • mqttPort: required(integer)
      • mqttTopic: required(string)

    HTTP status code 201

    Body

    Media type: application/json

    Type: object

    Properties
    • status: required(one of CREATED, RUNNING, COMPLETE, ERROR)
    • id: required(string)

    HTTP status code 400

    Body

    Media type: application/json

    Type: object

    Properties
    • status: required(one of CREATED, RUNNING, COMPLETE, ERROR)
    • description: required(string)
